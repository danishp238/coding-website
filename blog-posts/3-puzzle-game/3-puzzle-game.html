üìÑ HTML Structure

The HTML uses Tailwind classes to set up a responsive layout, a navigation bar with a logo, and two sections: one for draggable puzzle pieces and one for drop zones. 

<section class="grid grid-cols-3 grid-rows-3" id="puzzle-pieces"></section>

üñºÔ∏è Puzzle Initialization (JavaScript)

This block dynamically creates 9 puzzle pieces and assigns them a specific part of the full image using background images. Each image is draggable. 

for (let i = 0; i < 9; i++) {
  const row = Math.floor(i / 3) + 1;
  const column = (i % 3) + 1;

  const cell = document.createElement('div');
  cell.classList = 'bg-white/40 min-h-[100px] w-full';

  const puzzleImage = document.createElement('div');
  puzzleImage.classList = 'h-[100px] w-full bg-center bg-no-repeat bg-cover';
  puzzleImage.setAttribute('draggable', 'true');
  puzzleImage.setAttribute('ondragstart', 'drag(event)');
  puzzleImage.id = `piece-${row}-${column}`;
  puzzleImage.style.backgroundImage = `url('/images/puzzle-image-split/row-${row}-column-${column}.jpg')`;

  cell.appendChild(puzzleImage);
  puzzle_pieces.appendChild(cell);
}

üì¶ Drop Zone Setup

The drop zone consists of empty grid cells that listen for drop events.

üß≤ Drag-and-Drop Logic

These functions manage how pieces are picked up and placed into drop zones. 

function drag(event) {
  event.dataTransfer.setData("text", event.target.id);
}

function dropOver(event) {
  event.preventDefault(); // Required to allow dropping
}

function drop(event) {
  event.preventDefault();
  const data = event.dataTransfer.getData("text/plain");
  const draggedElement = document.getElementById(data);
  event.target.appendChild(draggedElement);
}
